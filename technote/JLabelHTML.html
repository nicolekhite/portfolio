Technote:
JLabels with HTML in Java
Author:
Nicole Kehaulani Hite
Summary:
This document explains how the JLabel works with HTML and how it can be implemented in Java.
Description:

JLabels are a Swing component in Java that is used to display strings and/or image icons. For this example of a JLabel, we'll be using the basic constructor as follows and adding it to the JPanel & JFrame:

// Label 1 - Basic JLabel
JLabel label = new JLabel("This is a simple test label");

By creating a JLabel and adding it to the JPanel and JFrame, the program will look like the following when run:


Now, JLabels have plenty of methods to change the font colors, size, and styles with the usage of Font objects. For example, the following code will set the font to change colors with the use of a Font object and JLabels methods.

// Adjust style of JLabel without HTML
label.setFont(new Font("Sans-Serif", Font.BOLD, 12));
label.setForeground(new Color(0x4287f5));

While this is simple, using HTML inside the JLabel will also accomplish this without the use of Font objects or method calls since it directly marks up the text itself.

To demonstrate the capabilities of using HTML within the JLabel, we can create a few more JLabels. This time, we will surround the string used in the constructor with <html> and </html> tags to indicate that the string contains HMTL.

// Label 2 - Setting up JLabel for HTML
JLabel label2 = new JLabel("<html>This is an HTML test label</html>")

This will look the same as the first label because we haven't added any tags to the HTML, keeping it very basic.

Now that we have a JLabel that is ready for HTML, we can add HTML tags within the <html>'s and the changes will be reflected when the program is run. You can find more HTML tags here, primarily in the section listed for "Text Formatting". For the next example, we'll use HTML to underlined the text in the JLabel. To do this, we'll create another JLabel and change the String within the JLabel constructor to have the tag for underlining (<u>) surround the String. By doing this, it will make the text underlined.

// Label 3 - Set up JLabel for underline HTML
JLabel label3 = new JLabel("<html><u>This is an underlined test label</u><html>");

Now that we know how the basics of using HTML in JLabels, we can add more complex HTML in these JLabels such as putting a table inside of the JLabel, or using in-line CSS to change the looks of the text.

Below is an example of putting a table into the JLabel with the basics used in label2 above with the <html>. The only difference is changing the original string to the code for a table, and adding in a border using some inline CSS to make it more visible.

// Label 4 - Table in HTML
String table = "<html><table border = '1'>"
    + "<tr>"
    + "<th>Name</th><th>Company</th><th>Position</th>"
    + "</tr>"
    + "<tr>"
    + "<td>Norm Niner</td><td>Apple</td><td>Software Engineer</td>"
    + "<tr>"
    + "<td>Norma Niner</td><td>Microsoft</td><td>CEO</td>"
    + "</tr>"
    + "</table></html>";
JLabel label4 = new JLabel(table);

As a result of the code above, the label will appear in the format of an HTML table below the other JLabels and look like the following:

As shown by the example above by adding the border, we can also add inline CSS to style our JLabels. A simple example can be shown below by the usage of <font>. In this example, we change the color of the text by surrounding the text with <font> and specifying the color of the font by doing <font color=orange> and closing the tag with </font>.

// Label 5 - Color with HTML
JLabel label5 = new JLabel("<html>This is a <font color=orange>color</font> test label</html>");
